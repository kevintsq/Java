## timeline

周三晚上 19:00 发布作业（每单元第一周同步发布训练）

周五中午 12:00 开启公测（每单元第一周同步发布训练答案）

周日晚上 22:00 公测提交截止，系统自动开始强测

周一中午 12:00 开始互测

周二晚上 22:00 关闭互测

周三中午 12:00 公布互测结果（一周后公布具体名单），开始bug修复（持续一周）
